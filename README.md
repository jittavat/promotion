# promotion
## Prerequisite
- Python 3.10
### Install Library
```commandline
pip install -r requirements.txt
```
## How to run
```commandline
# put tuple to main function between "00001" - "01999"
main(('00500', '00500', '00500', '00500', '00500'))
```
### Run
```commandline
python main.py
```
